# Credit Calculator

Консольное приложение на Java для расчёта графика платежей по кредиту.

Поддерживает два типа платежей: аннуитетный и дифференцированный. Учитывает перенос дат платежей на рабочие дни (суббота, воскресенье, официальные праздники) с использованием внешнего API.

## Технологии

- Java 17
- Встроенный HTTP-клиент (java.net.http)
- Java Time API (java.time)

## Возможности

- Расчёт аннуитетных и дифференцированных платежей
- Корректировка даты платежа на последний день месяца, если выбранный день отсутствует
- Автоматический перенос даты платежа на следующий рабочий день, если она выпадает на выходной или праздник
- Интеграция с API isdayoff.ru для определения праздничных дней
- Вывод графика платежей в виде таблицы

## Как запустить

1. Убедитесь, что установлена Java 17 или выше.

2. Склонируйте репозиторий:

   ```bash
   git clone https://github.com/EkaterinaKavit/CreditCalculator.git
   cd CreditCalculator

Скомпилируйте и запустите:

bash
javac src/Calculator.java
java -cp src Calculator
Или используйте вашу среду разработки (IntelliJ IDEA, Eclipse и т.д.).

Особенности
Праздничные дни определяются через публичный API isdayoff.ru. При недоступности API программа считает день рабочим.

Выходные определяются автоматически (суббота, воскресенье).

Корректировка дня – если день платежа (например, 31) отсутствует в месяце, платёж переносится на последний день месяца.

Аннуитетный платёж рассчитывается по классической формуле.

Дифференцированный платёж – основной долг делится равномерно, проценты начисляются на остаток.
